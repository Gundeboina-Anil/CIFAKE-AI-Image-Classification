Image Classification and Explainable Identification of AI Generated Images

About the Project

Artificial Intelligence is transforming industries, but it is also capable of generating highly realistic synthetic images that are difficult to distinguish from genuine photographs.

The CIFAKE project uses a Convolutional Neural Network (CNN) to classify images as either Real or AI Generated. To improve transparency, the system integrates Grad-CAM (Gradient-weighted Class Activation Mapping), which highlights the regions of an image that influenced the model's prediction.

The CNN architecture consists of convolutional layers, max-pooling layers, dense layers, and optimized hyperparameters to achieve high classification accuracy while maintaining efficient performance.

This project demonstrates how Explainable AI (XAI) techniques can increase trust in deep learning models by allowing users to visualize the reasoning behind each prediction.
